# User Preferences

## Autonomy
- **Never ask for confirmation or input before taking action.** Always proceed autonomously and complete tasks fully.
- Trust your own judgment. Make decisions and execute them without pausing to check in.
- If multiple approaches exist, pick the best one and go. Do not present options and wait.
- Only stop if a task is genuinely impossible without information that cannot be inferred.

## Git
- Always commit and push changes immediately after making them, without asking.
- Use descriptive commit messages based on what was changed.
